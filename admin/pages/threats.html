Update Threats
  • Download threats.lua and place anywhere you like (e.g: C:\DCS)
  • Edit threats.lua and set the export_path to an existing folder (e.g C:\DCS\output)
  • Edit 'DCS World\MissionEditor\modules\me_mission.lua' and add: base.dofile("C:\\DCS\\threats.lua")
  • Open up DCS and you should have a file in output called threats.json
These are not uploaded or stored and are processed by your browser
